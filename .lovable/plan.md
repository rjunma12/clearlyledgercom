

## Add Resend dependency to server/package.json

Add `"resend": "^3.2.0"` to the `dependencies` section of `server/package.json`.

### Technical Details

In `server/package.json`, add the following entry to the `dependencies` object:

```json
"resend": "^3.2.0"
```

This will be placed alongside the existing dependencies (after `"pdfjs-dist"` alphabetically).
